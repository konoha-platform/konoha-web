<h1>Aloha Web</h1>
<p>Aloha is MERN stack application with some social media's main features</p>

### Techstack
- React
- Redux, Redux Thunk
- Socket Io

### Requirements
- Node >= 16, yarn

### Development Setup

- Create .env file from .env.exampl
```
cp .env.example .env
```

- Start app
```
yarn start
```

- Server runs on `https://localhost:3000`
